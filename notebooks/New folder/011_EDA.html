01 - Exploratory Data Analysis (EDA) for HR Attrition¶

Objective¶

The primary goal of this notebook is to perform a comprehensive Exploratory Data Analysis (EDA) on the HR dataset. This involves:

  • Understanding the dataset's structure, distributions, and relationships.
  • Identifying missing values, outliers, and potential data quality issues.
  • Generating key performance indicators (KPIs) and visualizations relevant to employee attrition.
  • Deriving initial insights that will inform subsequent data cleaning, feature engineering, and model development.

In [7]:
if not df.empty:
    # Ensure 'Attrition' is a binary column (0/1 or 'Yes'/'No')
    # If 'Attrition' is string 'Yes'/'No', convert it to numerical 1/0 for easier calculation
    df['Attrition_Numeric'] = df['Attrition'].map({'Yes': 1, 'No': 0}) if 'Attrition' in df.columns and df['Attrition'].dtype == 'object' else df['Attrition']

    if 'Attrition_Numeric' in df.columns:
        overall_attrition_rate = df['Attrition_Numeric'].mean() * 100
        print(f"### Overall Attrition Rate: {overall_attrition_rate:.2f}% ###\n")

        attrition_by_features = ['Department', 'JobRole', 'Gender', 'EducationField', 'MaritalStatus'] # Add more as relevant

        for feature in attrition_by_features:
            if feature in df.columns:
                attrition_rate_by_feature = df.groupby(feature)['Attrition_Numeric'].mean().sort_values(ascending=False)
                print(f"Attrition Rate by {feature}:")
                display(attrition_rate_by_feature.to_frame(name='Attrition Rate (%)').style.format('{:.2%}'))

                plt.figure(figsize=(10, 6))
                sns.barplot(x=attrition_rate_by_feature.index, y=attrition_rate_by_feature.values * 100, palette='coolwarm')
                plt.title(f'Attrition Rate by {feature}')
                plt.ylabel('Attrition Rate (%)')
                plt.xticks(rotation=45, ha='right')
                plt.tight_layout()
                plt.show()
    else:
        print("Attrition column not found or not in expected format for rate calculation.")
